Decoración personalizada

Un montaje pensado para cada familia

Cada comunión se prepara de forma personalizada, adaptando la decoración al lugar, los colores, la temática y el tipo de celebración. Podemos crear una mesa principal, un photocall, una zona de regalos, una mesa dulce o una ambientación completa para que todo se vea bonito, ordenado y especial.

  • Mesa principal decorada.
  • Photocall o fondo para fotos.
  • Globos y detalles personalizados.
  • Mesa dulce o zona de chuches.
  • Decoración de entrada o rincón especial.
  • Montaje adaptado al espacio.

Qué podemos preparar
