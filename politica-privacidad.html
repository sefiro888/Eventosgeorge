Política de privacidad

Información básica sobre cómo se usan los datos que envías para solicitar información o presupuesto.

Responsable

Responsable del tratamiento: Eventos George. Contacto: eventosgeorge73@gmail.com y 666 21 35 03.

Datos que se pueden solicitar

Nombre, teléfono, email, fecha del evento, tipo de servicio, ubicación, número aproximado de invitados y detalles necesarios para preparar una propuesta.

Finalidad

Los datos se usan únicamente para responder consultas, preparar presupuestos, organizar servicios solicitados y mantener la comunicación relacionada con el evento.

Legitimación

La base legal es el consentimiento del usuario al enviar el formulario, escribir por WhatsApp, llamar o enviar un email solicitando información.

Conservación

Los datos se conservarán durante el tiempo necesario para gestionar la consulta o relación comercial, salvo que el usuario solicite su eliminación.

Cesión de datos

No se cederán datos a terceros salvo obligación legal o cuando sea imprescindible para prestar un servicio solicitado por el cliente.

Derechos

Puedes solicitar acceso, rectificación o eliminación de tus datos escribiendo a eventosgeorge73@gmail.com.

Servicios externos

La web enlaza con WhatsApp e Instagram. Al usar esos servicios, se aplican también sus propias políticas de privacidad.
